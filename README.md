# i2c-fundamentals
Introductory guide to Inter-Integrated Circuits (I²C) fundamentals using python.

## How to use:

1. git clone the repository on your machine and navegate to it:
```
git clone https://github.com/jhgpereira/i2c-fundamentals
cd i2c-fundamentals
```

2. Create a new python virtual environment and active it:
```
python -m venv /path/to/new/virtual/environment
source <venv>/bin/activate
```

3. Install the requirements:
```
pip install -r requirements.txt
```

4. Run the notebook:
```
marimo run i2c-fundamentals.py
```
